Welcome to Unilay

What is Unilay?

Unilay is the Universal Web Layout used for online apps at sensworks.ca. It provides a consistent user experience across apps, and removes the need to create a new design for every project - redirecting the focus to creating a useful app!

The main function of Unilay is to stylize default HTML elements such as links (<a>), headers (<h1>, <h2>, <p>, etc), and form elements (<input type="text">, <input type="submit">, etc). This particular iteration is made for use with Python frameworks like Flask and Django. To see a template containing all of the stylized elements, click here.

Unilay is easily customizable if you have a bit of HTML and CSS knowledge. However, a simple editor is in the works so that anyone can tweak Unilay to their needs.

Usability

Unilay 3.2 is responsive to the browser's size. Using CSS media queries, it switches to a single column view and uses larger fonts when viewed in a small window or mobile device. The media query breakpoint is 1000px - the landscape orientation on iPads will show the desktop version, and the portrait orientation will show the mobile version.

Error Handling

An error page is included (and programmed into) the Flask version. You can check it out here.

Potential Issues

Some technologies used in Unilay may not be compatible with older browsers. The current version uses CSS Grid (Check Browser Compatibility) for page layout and relies on Javascript for the mobile menu. In order to minimize the impact on browsers that do not support Javascript, please also include navigation links in the sidebar.
